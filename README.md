Identifica los 3 grupos de interés clave y los 3 temas materiales (ej. eficiencia energética del código, accesibilidad universal, privacidad de datos) para una consultora web.)
-Investigadores y Especialistas en Computación: Usuarios de repositorios como arXiv.org y Semantic Scholar para acceder a investigaciones de vanguardia en IA y Big Data.
-Desarrolladores y Arquitectos de Software: Profesionales que consultan bases de datos como DBLP y DOAJ para encontrar publicaciones sobre Desarrollo Web y Multiplataforma.
-Académicos del ambiente español: Estudiantes y docentes que utilizan Dialnet para investigar la aplicación de tecnologías en el contexto de España y Latinoamérica.
-Inteligencia Artificial y Modelos de Lenguaje: Considerada investigación de vanguardia esencial, consultable en plataformas impulsadas por IA como Semantic Scholar.
-Ingeniería de Software y Web: Tema fundamental verificado a través de directorios como el DOAJ, enfocado en la calidad de las publicaciones técnicas.
-Big Data y Análisis de Datos: Área crítica para encontrar monografías y libros de texto especializados en el Directory of Open Access Books (DOAB).
